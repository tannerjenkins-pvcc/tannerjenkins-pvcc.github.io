Chapter 3 Operating Systems

TERM

DEFINITION
operating systemThe most important software in a computer. It manages memory, along with all of it's processes, software, hardware, and communication. Without an operating system, the user cannot communicate with the computer.
job queContains and ordered list of jobs or actions for the subsystem to process and execute.
FIFOStands for First In, First Out. It essentially means that the first file to be stored is the first one to be retrieved. This means the oldest file is the first one to be used.
interactive processingThe user needs to provide instructions while the computer is processing.
time-sharingMethod of operation where multiple users with several different programs can interact at the same time with the CPU of a larger digital computer.
multiprogrammingForm of parallel processing where several programs can run all at once on one singular CPU system.
application softwareTyper of program that carries out a specific function. THis can be for any purpose, whether personal, business, educational, etc.
system softwareRuns the hardware of the computer, managign all basic functions to facilitate the more complicated ones.
utility softwareOptimization software that helps the operating system organize data and manage disc space along with all basic functions.
graphical user interface (GUI)Digital interface where the user operates with graphical compnents like icons, buttons, menus, etc.
shellProgram that exposes the operating system's services to a person or another another program for service.
kernelThe core program for all other parts and programs for the system.
device driverSoftware program without a user interface that manages hardware components or or peripherals attached to a computer and enables them to function with the computer smoothly.
file managerOrganized stucture or space in a system where information is easy to retrieve.
memory managerAllocates portions of memory programs and free it for reuse when space is free.
virtual memoryMemory management technique where secondary memory can be used as if it was an extension of the main memory.
boot strappingComputer language compilers that are able to be coded in the same language.
ROMStands for Read Only Memory. It is a type of storage that can only be read, not altered in any way. It alows the computer to start up or regenerate when turned on
deadlockSituation where a set of processes are blocked because each process is holding a resource and are waiting for another resource.
spoolingData is temporarily held to be used and executed by the device, program, or the system.
super user (or administrator)Highly privelaged accounts in a system that have more access to more features than the average users. This is primarily used by administration or IT employees.
multi-factor authenticationA multi-step account sign in process that requires more than a password. Sometimes requiring more than one computer to sign in.
